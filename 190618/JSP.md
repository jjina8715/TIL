# JSP

1. JSP 태그

   - 스크립트 태그 : 필요한 자바 코드를 정의하는 용도의 태그들
     - <%= > : 표현식
     - <% %> : 수행문, 지역변수 선언
     - <%! %>  : 선언문, 멤버변수 선언, 메서드 정의
     - <%@ %> : 지시자
     - <%-- --%> : 주석(HTML 주석 : &lt;!-- -->, 자바 주석 : //, /* */)
   - 액션 태그
     - <jsp:useBean .../>
     - <jsp:forward .../>
     - <jsp:include .../>
     - <jsp:setProperty .../>
     - <jsp:getProperty .../>
   - 커스텀 태그
     - 필요에 의해 개발자가 직접 태그를 만들어서 사용
     - JSTL : 톰캣의 개발자들이 만들어서 배포한 커스텀 태그
