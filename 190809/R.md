# R

[빅데이터]

- IT컨설팅업체인 가트너는 빅데이터를 3V, 즉 대용량의 데이터 규모(High-Volume), 빠른 입출력 속도(High-Velocity), 다양성(High-Variety)으로 정의
  - 데이터의 기하급수적 성장, 데이터의 가용성, 데이터의 정보화 관점에서 내린 정의
- 데이터 과학 : 빅데이터를 분석하고 연구하는 학문
- 데이터 과학자 : 데이터에서 의미 있는 정보를 도출하는 사람

[R]

- 통계 분석에 특화된 언어

- 뉴질랜드 오클랜드 대학의 로버트 젠틀맨과 로스 이하카가 개발한 언어

- 장점

  - 통계 분석 등에 활용할 수 있는 패키지가 많다.
  - 사용자 커뮤니티가 활성화되어 있어 다양한 정보 공유 가능
  - 무료로도 이용 가능
  - 설치와 사용환경 구축이 간편
  - 맥, 윈도우, 리눅스 등 다양한 운영체제에서 동작
  - 다른 프로그래밍 언어에 비해 한글 처리가 쉽다

- 단점

  - 범용 프로그래밍 언어에 비해 처리 속도가 느리다
  - 데이터 분석에만 특화되어 있다 보니 대규모 IT 서비스 개발에 접목하기 어렵다
  - 문제가 발생했을 때 스스로 해결

- 설치

  - [R](https://www.r-project.org)

  - [RStudio](https://www.studio.com)
